## How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Run backend:
python app.py

3. Run UI (in another terminal):
streamlit run ui.py

4. Open browser:
http://localhost:8501

## Features

- Data ingestion via API
- Hash-based deduplication
- Fault-tolerant handling
- Aggregated results view
- Simple Streamlit UI
